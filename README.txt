MAPA MAESTRO DEL DESTINO - PROYECTO WEB

Qué incluye:
- Sitio web en Next.js con Tailwind
- Formulario para ingresar datos natales
- Conexión al modelo GPT-4 con prompt estilo Saullo
- Generación automática de informe PDF
- Entrega por descarga directa
- Preparado para integración con USDT

Pasos para usar:
1. Subí este proyecto a un nuevo repositorio de GitHub.
2. En Vercel, importá el repositorio y seguí las instrucciones.
3. Configurá la clave de OpenAI como variable de entorno: OPENAI_API_KEY
4. ¡Probalo y salí a romperla!
